# Generator_password
Данная программа была написана для закрытия потребностей в криптостойких паролях внутри организации.
Проведено тестирование и сборки под разные компьютерные платформы,критических нарушений в работе не выявлено.

Для Windows есть готовая сборка работающая из коробки.

Для запуска исходника на своем компьютере вам потребуются следующие пакеты и программы:

PS инструкция для Linux ubuntu(deb)
- python3 3.10.6
- sudo apt-get install python3-tk 
- pip install tk (подразумевается, что pip у вас стоит)   

Теперь вы можете скопировать и доработать под свои нужды файл:
- Tkinter_UI.py
